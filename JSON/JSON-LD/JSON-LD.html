JSON-LD

JavaScript Object Notation for Linked Data (JSON-LD) is described as JavaScript code rather than markup elements and attributes. 

				demo1:

				<script type="application/ld+json">
				{
				  "@context": "http://schema.org",
				  "@type": "Person",
				  "image": "johnsmith.jpg",
				  "name": "John Smith",
				  "url": "http://www.johnsmith.com"
				}
				</script>
			
				demo2:


				[
				  {
				    "@type": [
				      "http://schema.org/Person"
				   ],
				    "http://schema.org/image": [
				      {
				        "@id": "http://www.johnsmith.com/images/johnsmith.jpg"
				      }
				   ],
				    "http://schema.org/name": [
				      {
				        "@value": "John Smith"
				      }
				   ],
				    "http://schema.org/url": [
				      {
				        "@id": "http://www.johnsmith.com"
				      }
				   ]
				  }
				]
