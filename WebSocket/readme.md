```js
var connection = new WebSocket('ws://html5rocks.websocket.org/echo', ['soap', 'xmpp']);

//请注意 ws:。这是 WebSocket 连接的新网址架构。
//对于安全 WebSocket 连接还有 wss:，就像 https: 用于安全 HTTP 连接一样。

```
