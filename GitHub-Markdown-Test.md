## Footnote
### _NOTE: Footnote syntax is only supported on server side._
```
    [^1]: This is a footnote
    [^label]: A footnote on "label"
    [^!DEF]: The definition of a footnote.
```
Footnotes[^1] have a label[^label] and a definition[^!DEF].
